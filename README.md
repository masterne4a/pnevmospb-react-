Миграция легаси-сайта: HTML/CSS → React & Vue

Исходный статический сайт (vanilla HTML/CSS) полностью переписан с использованием двух популярных фреймворков:

React-версия: Компонентный подход, управление состоянием.

Vue-версия: Декларативная логика, удобный синтаксис.

В ходе работы сохранена оригинальная стилистика и семантика, но код стал более структурированным, переиспользуемым и масштабируемым.
